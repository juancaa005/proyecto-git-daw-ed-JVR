# Mini Proyecto Git - DAW

Este es un mini proyecto creado para **practicar Git y GitHub**, usando commits frecuentes, ramas y Pull Requests.

---

## Contenido

### `src/Main.java`
- Clase principal en Java.
- Incluye el método `main()` y un método `saludar()` de ejemplo.

### `web/index.html`
- Página HTML básica para probar cambios.

### `web/styles.css`
- Estilos CSS simples para el HTML.

---

## Ramas

- `feature-java-utils`: cambios en Java
- `feature-web-layout`: cambios en HTML y CSS
- `hotfix-readme`: correcciones en el README
